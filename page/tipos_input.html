Input

Atributos do input tipo text

readonly - Apenas leitura

disabred - campo desabilitado

size - tamanho do campo, largura do campo

maxlength - quantidade de caracteres total

required - campo obrigatório

autofocus - define que o campo terá foco quando a página for carregada

autocomplete = 'on' - quer dizer que será autocompletado com valores já digitados no navegador anteriormente

list="minha lista" - lista em um input usando datalist com mesmo id e as options dentro da lista

placeholder - Já mostra preenchido um valor exemplo no campo input

pattern - determina um padrão atravez de uma expressão regular

input password - tipo senha

input submit - envio dos dados

input type reset - apaga o formulário

inpup radui - seleção única, quando o name é igual, ai a seleção é única
